College Finder

Description
College Finder is a web-based application designed to help students and parents easily search and view college information from a single platform.
It simplifies the college selection process by providing centralized, clear, and accessible college details for informed decision-making.



Features:
Search colleges based on basic criteria.
View detailed college information.
Simple and user-friendly interface.
Fast data retrieval from backend/API.
Responsive design for different devices.



Tech Stack:
React.js – Frontend development.
Tailwind CSS – Styling and responsive UI.
Gemini API – AI support for data/content assistance.



Installation:
npm install
npm run dev


Usage:
Open the application in a browser.
Browse or search for colleges.
Click on a college to view detailed information.
Compare options and choose the best-fit college.
